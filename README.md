# 📖 Crypto Metrics <a name="about-project"></a>

**Crypto Metrics** is a single page web application that display a list of cryptocurrency coins and their current market info. it allows users to search for coins and filter it by it's names or symbols.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:
- Browser
- Code Editor
- Node.js

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/sahlminkok/CryptoMetrics.git
```

### Install

Install this project with:

```sh
  cd CryptoMetrics
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```

### Run tests

To run tests, run the following command:

```sh
  npm test
```

### Deployment

You can deploy this project using:

- GitHub pages.
- Render.
- Netlify.
